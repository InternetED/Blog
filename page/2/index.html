一、Java對象生命週期

Java代碼編譯後生成的字節碼.class文件從文件系統中加載到虛擬機之後，便有了JVM上的Java對象，Java對象在JVM上運行有7個階段，如下：

  • Created（創建）
  • InUse（應用）
  • Invisible（不可見）
  • Unreachable（不可達）
  • Collected（收集）
  • Finalized（終結）
  • Deallocated（對象空間重新分配）

1、Created（創建）

  1. Class 對象首次加載的時候進行一次，有關靜態初始化的所有動作從超類到子類會依序執行。
  2. 在堆上為對象分配足夠的存儲空間。
  3. 存儲空間會被清空，也就是說對象中的所有基本類型數據都為空
  4. 從超類到子類依序初始化成員變量。
  5. 從超類到子類依序執行構造方法。
