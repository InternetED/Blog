創建 SSH KEY

在 Windows 上創建 SSH KEY

  1. 檢查現有的 SSH KEY

    開啟命令提示字元，然後執行：

    cd %userprofile%/.ssh

    • 如果顯示 “系統找不到指定的路徑。”就前往下一步
    • 反之，則使用先前創建好的 SSH KEY

  2. 創建 SSH KEY

    在命令提示字元中執行：

    ssh-keygen -t rsa -C "Your's email@example.com"

    注意！ssh-keygen 命令僅在已安裝Git時可用

    在 Linux 上創建 SSH KEY

  3. 檢查現有的 SSH KEY

    開啟終端機，然後執行：

    cd ~/.ssh

    • 如果顯示 “系統找不到指定的路徑。”就前往下一步
    • 反之，則使用先前創建好的 SSH KEY

  4. 創建 SSH KEY

    在終端機中執行：

    ssh-keygen -t rsa -C "Your's email@example.com"
